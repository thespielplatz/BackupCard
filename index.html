Extended Public Key - Read Only Zugriff

The following data is not security-critical. However, it allows for the identification of all Bitcoin addresses associated with the wallet, determining the current balance of the wallet, and viewing all transactions conducted so far.

The Extended Public Key can be used with compatible wallets to obtain a read-only live view of the wallet and to determine (new) addresses in case further deposits are to be made to this wallet.

The Extended Public Key should, therefore, be kept confidential.
